# LowPower-ALU-FSM
FSM-based Low Power ALU using SystemVerilog and Vivado
Low Power FSM-Controlled ALU
Project Overview
This project implements a low-power Arithmetic Logic Unit (ALU) using a Finite State Machine (FSM) to reduce dynamic power consumption by controlling switching activity.
**Features**
* 8-bit ALU (ADD, SUB, AND, OR, XOR, SHL, SHR)
* FSM-based power control
* Output gating for low power
* Simulation using Vivado
* Power analysis included
**Design Architecture**
* ALU → Datapath
* FSM → Control logic
* MUX → Power gating
## Results
* Reduced dynamic power
* Verified using Vivado waveform
* Power report analysis included
## Tools Used
* Xilinx Vivado
* SystemVerilog / Verilog
## Outputs
* RTL Schematic

* Power Report
